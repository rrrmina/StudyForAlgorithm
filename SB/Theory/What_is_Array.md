# Array

---

동일한 자료형(Data Type)의 데이터를 연속된 공간에 저장하기 위한 자료구조

```java
DataType[] variable = {data1, data2, ... };
```

```java
DataType[] variable = new DataType[size];
variable[0] = data0;
variable[1] = data1;
...
```

```java
    for (DataType data : variable) {
      // 향상된 for 문으로 array data 접근 가능
    }
```
